# Yuiel LP 画像差し替えマップ

画像ファイルを以下のファイル名で `~/yuiel-lp/images/` に保存してください。
ファイル名を合わせるだけで自動的にサイトに反映されます。

## 注意事項
- 形式: WebP推奨（JPG/PNGも可）
- JPG/PNGの場合はファイル名の拡張子を変えてHTMLも合わせて修正
- 画像は自動でobject-fit: coverで表示されるため、多少サイズが異なってもOK
